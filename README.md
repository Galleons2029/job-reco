# job_reco

职位推荐和用户画像系统

## 目录介绍

```coffeescript
.
├── app             # 项目代码
│   ├── agent_HR    # HR端
│   ├── crawler     # 爬虫
│   ├── db          # 数据库
│   └── pipeline    # 数据处理
└── tests           # 测试目录

```

## 快速开始

使用 **TOX** 构建，支持 **CI** 跨平台，跨版本测试。测试可以使用TOX 自动完成

1. 安装TOX
    ```bash
    pip install tox
    ```

2. 测试代码
    ```bash
    tox
    ```
